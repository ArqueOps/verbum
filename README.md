# Verbum — Word for All

Plataforma de estudo biblico com IA.

## Stack
- **Frontend**: Next.js + TypeScript
- **Backend**: Supabase (Auth, DB, Storage)
- **IA**: OpenAI (estudo biblico contextualizado)
- **Deploy**: Vercel
- **Pagamentos**: Caramelou (BR)

## Status
